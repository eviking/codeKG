Add Repository

Repos must be accessible inside the Docker container at the path you specify. Mount your repo under {{ repos_path }} on the host — e.g. docker-compose.yml REPOS_PATH=/path/to/your/repos.

Host Setup

Set REPOS_PATH in your .env to the directory on your host that contains the repos you want to index. Each subdirectory becomes a mountable path. 

# .env
REPOS_PATH=/Users/yourname/code

# Then repos are accessible as:
# /repos/my-service   (from /Users/yourname/code/my-service)
