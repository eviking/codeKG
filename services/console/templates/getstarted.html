4

Connect the MCP server to Claude Code

Run one command — then AI agents can call codeKG tools directly

Run this command

Run this once in your terminal to register codeKG as an MCP server in Claude Code. Use ! command in the Claude Code prompt to run it directly in this session.

Add MCP server
claude mcp add codekg --transport stdio -- docker exec -i codekg-mcp python main.py
Verify it's registered
claude mcp list # Should show: codekg stdio docker exec -i codekg-mcp python main.py
Test it — ask something about your codebase
# Open a Claude Code session inside your indexed repo and ask: What are the highest blast-radius classes in this codebase?
